自己紹介

東京を拠点に活動するWebエンジニアです。5年以上のフロントエンド開発経験と3年のバックエンド開発経験を持ち、モダンなWebアプリケーション開発に情熱を注いでいます。

ユーザー体験を最優先に考え、クリーンで保守性の高いコードを書くことを心がけています。新しい技術にも積極的に取り組み、チームでの開発経験も豊富です。

趣味はテックカンファレンスへの参加と、オープンソースプロジェクトへの貢献です。日々の学びを大切にし、常にスキルアップを目指しています。
